🖼️ Processador de Imagens

📤 Upload Original

Faz upload da imagem sem processamento


📏 Redimensionamento

Redimensiona a imagem para 300x200 pixels


🗜️ Compressão

Comprime a imagem mantendo 50% da qualidade original


⚡ Processamento Completo

Redimensiona para 500x300 pixels + comprime com 70% de qualidade


🔄 Conversão para JPG

Converte qualquer imagem para formato JPG com compressão otimizada


🔄 Conversão para PNG

Converte qualquer imagem para formato PNG (sem perda de qualidade)


🔄 Conversão para WebP

Converte qualquer imagem para formato WebP (máxima compressão)


📋 Como usar:

  1. Escolha o tipo de processamento desejado
  2. Selecione e escolha uma imagem (jpg, png, webp)
  3. Clique no botão correspondente
  4. A resposta JSON conterá a imagem processada em base64
